1. Configure api key to access the APIs of https://petstore.swagger.io/v2/swagger.json. Follow the steps:
		a. Open https://petstore.swagger.io/v2/swagger.json.
		b. Click on "Authorise", then enter a value in "Value" field and click "Authorise" button.
		c. Configure the same value in "ApiKey" in App.config file.
2. UnitTest1.cs has four tests covering CRUD operations with pet.
3. Run tests from test explorer.
